Frequently Asked Questions

  1. If the Beneficiary Key is lost, what do I do?

    2. If either key is lost, the seed phrase CANNOT be decrypted. This is the risk of self custody. Unless there are other means of retrieval, the Bitcoin is gone. Legacy does not save any data.

  2. Can I create multiple key-pairs?

    3. Yes. You can create as many key-pairs you like. They can be associated with as many different seed phrases as you like.

  3. How do I use the Legacy software offline?

    4. Download "Legacy-offline.html" from gitbub. Disconnect from wifi. Open the file in a web browser.

  4. Are there any restrictions on allowable characters for the keys?

    5. No. Beneficiary Key and Benefactor Key may contain all uppercase, lowercase, numbers, and special characters.

  5. What information needs to be included in the Deadman Switch Email?

    6. The Beneficiary needs to receive:

    • Benefactor Key
    • Encrypted Seed Phrase
    • Working copy of Legacy-offline.html
  6. How exactly does the encryption protocol work?

    7. Visit the Protocol page for detail summary.
