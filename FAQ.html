Frequently Asked Questions

    If the Beneficiary Key is lost, what do I do?

  • If either key is lost, the seed phrase CANNOT be decrypted. This is the risk of self custody. Unless there are other means of retrieval, the Bitcoin is gone. Legacy does not save any data.

    • Can I create multiple key-pairs?

  • Yes. You can create as many key-pairs you like. They can be associated with as many different seed phrases as you like.

    • How do I use the Legacy software offline?

  • Download the encrypt.html, decrypt.html and english.txt files from the github repository. In a terminal window, navigate to the folder containing the files. Disconnect from Wifi, internet, etc. Run the command python3 -m http.serverand then open a browser window and go to http://localhost:8000 The page will load without internet connection because it is hosted on a local server on your computer.
